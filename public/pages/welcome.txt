# Welcome

Welcome.

This is a small collection of projects, ideas, and things I'm building. Every page is nothing more than a plain text file. Open one in any editor, make a change, save it, and the site updates. Software has a habit of becoming more complicated than it needs to be, so this doesn't.

===

## How it works

Every .txt file in the pages folder becomes a page here. The filename becomes the page's address.

The order in the table of contents is controlled with a single line at the top of each file. The example below is shown literally; the leading space is only there so it displays as text instead of being read as a real header:

 # order: 1

Smaller numbers appear first. If no order is given, pages are sorted alphabetically.

===

## The link command

To add a button, write one line like this. The leading hash is only there so the example displays as text instead of turning into a real button:

#/link(Button label example.com)

The text before the URL becomes the button label. It renders as a button with a small oxblood tick and a subtle URL stamp beside it. Plain URLs written in the text also work and appear as quiet inline links.
